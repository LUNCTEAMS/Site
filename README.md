# LUNC TEAMS — Red vs Blue (GitHub Pages)

Static site (no build tools). This pack uses the **original cinematic design** + your requested copy changes.

## Open locally
- Open `index-inline.html` (recommended). It cannot “lose” CSS/JS.
- Or open `index.html` (requires `styles.css` + `script.js` to be in the same folder).

## GitHub Pages deploy
Upload these to your repo root:
- `index.html`
- `styles.css`
- `script.js`
- `index-inline.html` (optional)
- `.nojekyll`
- `assets/` folder

Then: **Repo → Settings → Pages → Deploy from a branch → main → /(root)**.

## Set your Telegram link
Search & replace in `index.html`:
`https://t.me/REPLACE_WITH_YOUR_TELEGRAM`
